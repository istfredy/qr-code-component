### What I learned

Nothing, but I can just tell you that to center the content of a dive horizontally and vertically in **CSS** , you can do the following :

```css
.class-of-parent-div-container{
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}
```
